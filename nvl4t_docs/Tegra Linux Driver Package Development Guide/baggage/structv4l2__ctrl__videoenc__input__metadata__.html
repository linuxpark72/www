v4l2_ctrl_videoenc_input_metadata_ Struct Reference
V4L2 Video Encoder

Detailed Description

Holds the encoder frame input metadata parameters.

Must be used with V4L2_CID_MPEG_VIDEOENC_INPUT_METADATA IOCTL.
